# Nathan Burdett Personal Portfolio

A simple static HTML website showcasing Nathan Burdett's professional profile as a Sales Leader.

## Project Overview

This is a personal portfolio website built with plain HTML and CSS. It includes sections for about, experience, and contact information.

## Files

- `index.html`: The main HTML file containing the website structure and content.
- `mountains.jpeg`: Background image shown behind the header.
- `profile.jpeg`: Personal photo shown in the About section.
- `game.html`: Separate page for the Reaction Click mini-game.
- `basketball.html`: Page showcasing Nathan's top 5 NBA players with a featured video.
- `nba.avif`: Background image for the basketball page header.

## How to View

Open `index.html` in any web browser to view the website.

## Technologies Used

- HTML5
- CSS3
